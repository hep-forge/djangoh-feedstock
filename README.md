Feedstock for [djangoh](https://github.com/spiesber/DJANGOH) — part of [hep-forge](https://anaconda.org/hep-forge).

DJANGOH simulates deep inelastic lepton-proton scattering for NC and CC events, including QED and QCD radiative effects

## Install

```bash
conda install -c hep-forge -c conda-forge djangoh
```
